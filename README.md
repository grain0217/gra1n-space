https://grain0217.xyz


### 说明
在`config.js`文件中里修改个人信息。

### 运行  

1. clone项目

2. 安装依赖
```
npm i
```

3. 在config.js中修改个人信息

4. 开发环境调试：
```
npm run dev
```

5. 构建静态页面
```
// 输出到public目录下
npm run build // 或者 gatsby build
```

静态资源构建完成后，可以选择Nginx托管。

如果你没有服务器，可以选择通过以下静态网站托管服务部署：
- [surge.sh](https://surge.sh/)
- [Netlify](https://www.netlify.com/)
- [now](https://vercel.com/home)
- ...
